# HNG Stage 1 Task - Profile Card

A simple, responsive, and accessible Profile Card built as part of the HNG Internship Stage 1 task. This static web page displays basic user information, social links, hobbies, dislikes, and a real-time current time clock in milliseconds.

## Features

- **Profile Information**: Displays user avatar, name, and a brief bio.
- **Real-Time Clock**: Utilizes JavaScript to show the current time in milliseconds, dynamically updating.
- **Social Links**: Provides quick access to Twitter and GitHub profiles.
- **Personal Details**: Lists hobbies and dislikes.
- **Responsive Design**: Adapts cleanly to both desktop and mobile screens for an optimal viewing experience.

## Technologies Used

- **HTML5**: Semantic tags used for structuring the content.
- **CSS3**: Custom vanilla CSS for styling, utilizing Grid, Flexbox, and CSS variables.
- **JavaScript**: Minimal JavaScript script to handle the real-time millisecond time updates.

## Getting Started

To view this project locally:

1. Clone this repository or download the source files.
2. Navigate to the project directory.
3. Double-click on `index.html` to open it in your preferred web browser.

*(Alternatively, if you use VS Code, you can right-click `index.html` and select **"Open with Live Server"**).*
